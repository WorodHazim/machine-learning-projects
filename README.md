# Machine Learning Projects Portfolio

This repository contains a collection of machine learning projects built while learning and practicing core ML concepts such as data analysis, preprocessing, model training, evaluation, and comparison.

Each project focuses on solving a real-world machine learning problem using Python and popular ML libraries.

---

## Projects Included

### 1. Titanic Survival Prediction
A machine learning project that predicts whether a passenger survived the Titanic disaster.

Key concepts:
- Exploratory Data Analysis (EDA)
- Data preprocessing
- Classification models
- Model comparison

Models used:
- Logistic Regression
- Decision Tree
- Random Forest

---

### 2. Heart Disease Prediction
This project predicts the likelihood of heart disease based on medical features such as cholesterol levels, chest pain type, and heart rate.

Key concepts:
- Correlation analysis
- Feature relationships
- Confusion matrix
- Model comparison

Models used:
- Logistic Regression
- Decision Tree
- Random Forest

---

### 3. Diabetes Prediction
This project predicts whether a patient has diabetes using medical attributes such as glucose level, BMI, and age.

Key concepts:
- Data cleaning
- Handling missing values
- Feature scaling using StandardScaler
- K-Nearest Neighbors (KNN)

Models used:
- Logistic Regression
- KNN
- Random Forest

---

### 4. House Price Prediction
This project predicts house prices using regression models based on features such as house size, number of bedrooms, and house age.

Key concepts:
- Regression problem
- Feature selection
- Train/Test split
- Linear Regression
- Mean Squared Error (MSE)

Models used:
- Linear Regression

---

## Technologies Used

## Machine Learning Workflow Used

Each project follows a standard machine learning workflow:

1. Data Loading  
2. Exploratory Data Analysis (EDA)  
3. Data Cleaning / Preprocessing  
4. Feature Scaling (when needed)  
5. Train/Test Split  
6. Model Training  
7. Model Evaluation  
8. Model Comparison
